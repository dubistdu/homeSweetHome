# About the App

#### Ruby v '2.3.3'

### real estate list app

## Features
- User can list a new house
- User need to have pictures to complete listing homes
- User login through 3rd party account such as github

  * refile

#### - Model
  * home
  * user

#### - Controller
  * home
  * session
